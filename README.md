# 💘 Dating App Behavior Analysis

## 📌 Overview
This project analyzes user behavior on dating applications using a dataset of 50,000 users and 19 behavioral + demographic features.  
The goal is to uncover patterns in user engagement, swipe behavior, messaging activity, and match outcomes through Exploratory Data Analysis (EDA) and data visualization techniques.
The analysis is performed in a Jupyter Notebook using Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn.

# 🛠️ Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

# 📊 Analysis Performed

## ✔ Exploratory Data Analysis (EDA)
- Dataset overview
- Missing value analysis
- Statistical summaries
- Data type inspection

## ✔ Behavioral Insights
- Swipe behavior analysis
- Match success patterns
- User engagement trends
- Messaging activity analysis

## ✔ Data Visualization
- Histograms
- Count plots
- Box plots
- Correlation heatmaps
- Pair plots

# 📈 Key Insights
- Users with longer bios and more profile pictures tend to receive more matches.
- Messaging activity positively impacts match success.
- Swipe behavior strongly influences mutual match rates.
- User activity timing affects engagement patterns.
